How to help out

This page exists to try to hook people up with soccer teams in their area. Basically, the biggest thing is making sure data is getting propperly published.

If you see problems report them, you can use github, email, or just yell really loud.

If you want to volunteer submit some pull requests or issues on github and we can get some things worked out.
